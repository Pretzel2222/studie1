Fase 3 - Webutvikling og UX

Fase 3 - Webutvikling og UX

Fase 3: Virkeliggjøre



Dette er siste fasen av webutvikling, og her handler det om å realisere det vi har jobbet med i fase 1 og fase 2. Her fokuserer vi på å kode, teste og feilsøke for å sikre at det endelige produktet fungerer. Dette markerer avslutningen av utviklingsprosessen før produktet blir lansert.

I begynnelsen bruker vi HTML, som står for "Hypertext Markup Language". Dette er et språk brukt til å utvikle nettsider. Det brukes for struktur og består av en rekke koder som angir forskjellige elementer som overskrifter, avsnitt, bilder, tabeller og lenker. CSS er hoveddelen for design. Her kan du lage alt fra enkelt design til avansert design ved å bruke forskjellige funksjoner som CSS tilbyr, inkludert farger, skrifttype, skriftstørrelse og alt som gjør nettsiden visuelt tiltalende.

CSS blir vanligvis skrevet inn i et eget dokument, men det kan også skrives i samme dokument som HTML. For større prosjekter er det smartest å ha et eget dokument for CSS og koble det til HTML-dokumentet. For å bruke HTML starter man som regel med "!DOCTYPE html". Dette vil sette opp alt du trenger av elementer for å starte kodingen på HTML.

Hvorfor lærer vi dette?



Å lære HTML og CSS er grunnleggende for webutvikling. Dette er en fase som alle utviklere går gjennom, men senere vil de også lære mer avanserte kodespråk. HTML og CSS gir evnen til å strukturere og designe en nettside på hvilken som helst måte. Her er det kun din kreativitet som kan stoppe deg. Disse to kodespråkene kan ta deg langt i webutviklerverdenen og er svært attraktive i dagens arbeidsmarked. Det gir deg også en uavhengighet der du kan jobbe med egne prosjekter og bygge hva du ønsker.
